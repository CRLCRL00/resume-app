### 3. 完成度爆花 (R119 A5)

```js
_triggerExplode() {
  const particles = [];
  const colors = ['#fbbf24', '#6366f1', '#8b5cf6', '#07c193', '#ec4899'];
  for (let i = 0; i < 60; i++) {
    const angle = (Math.PI * 2 * i) / 60 + Math.random() * 0.3;
    const dist = 200 + Math.random() * 300;
    particles.push({ id: i, dx: Math.cos(angle) * dist, dy: Math.sin(angle) * dist, color: colors[i % colors.length] });
  }
  this.setData({ showExplode: true, explodeParticles: particles });
  setTimeout(() => this.setData({ showExplode: false, explodeParticles: [] }), 1500);
  if (wx.vibrateShort) try { wx.vibrateShort({ type: 'medium' }); } catch (_) {}
}
```

wxml 粒子动画:
```xml
<view class="explode-overlay" wx:if="{{showExplode}}">
  <view wx:for="{{explodeParticles}}" class="explode-particle" style="--dx: {{p.dx}}rpx; --dy: {{p.dy}}rpx; background: {{p.color}};"></view>
</view>
```

效果: 100% 时 60 颗粒子从中心炸开, 5 色随机方向, 振动反馈, 1.5s 自动隐藏.

### 4. 进度提示动态 (R119 B2)

```js
_updateProgressPrompt(completion) {
  let bucket = -1; let text = ''; let hint = '';
  if (completion >= 100) { bucket = 4; text = '🎉 全部完成！'; hint = '可以一键投递了'; }
  else if (completion >= 75) { bucket = 3; text = '💪 只差一点点！'; hint = '再加把劲就完成'; }
  else if (completion >= 50) { bucket = 2; text = '🚀 完成一半啦！'; hint = '再填几个字段就过 50%'; }
  else if (completion >= 25) { bucket = 1; text = '✨ 不错的开始！'; hint = 'AI 正在帮你梳理'; }
  else if (completion > 0) { bucket = 0; text = '🎯 开始你的简历之旅'; hint = '跟随 AI 助手一步步来'; }
  if (bucket !== this.data._lastCompletionBucket) {
    this.setData({ progressPromptText: text, progressPromptHint: hint, _lastCompletionBucket: bucket });
  }
}
```

效果: 跨档才更新文案 (0% → 25% → 50% → 75% → 100%), 避免每次填都刷屏. 像游戏进度感.

## 踩坑笔记

### Nit #1: R107 翻转测试与 R119 explode-particle 同前缀冲突

**问题**: R114 T3 测试断言 `!wxss.includes('@keyframes explode')` 防 R107 庆祝 keyframe 重新引入. R119 A5 加了 `@keyframes explode-particle` (粒子爆花), 字符串前缀 explode 触发误报.

**解决**: 用 regex `/@keyframes explode[^-\w]/` 精确匹配 — explode 后必须跟非字母数字 (排除 explode-particle).

### Nit #2: R116 翻转测试与 R119 avatar-breathe 同关键词

**问题**: R116 测试断言"wxss 全部星图 keyframes 已清", 但 R119 加了 avatar-breathe / avatar-celebrate (新动效, 非星图).

**解决**: 翻转测试加注释说明 R119 仍可保留这些动效 keyframes, 重点是 R107 旧装饰 (meteor-fall / spin-slow / num-pulse / constellation-breathe) 已删.
