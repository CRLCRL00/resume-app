# 开发日志 — 2026-07-17（Phase 9 Round 106 — token 缺失时 UI 提示）

> 前置：[2026-07-17-phase9-round105.md](../devlog/2026-07-17-phase9-round105.md)

## 起点

R98-R105 真机 preview 反复 401 missing token:
- 用户 IDE Console 跑 fetch dev-reissue 拿到 token
- 但 token 在提交时丢失 (storage GC / 进程重启 / 真机 IDE)
- 用户不知道怎么办

**根因**: 用户手动跑 fetch 不持久 (storage 清掉就丢). 短期 UI 提示 + 永久 fix (server 加 `/auth/refresh`) 都需要.

### 关键代码
```js
onSubmit() {
  const token = wx.getStorageSync('token');
  if (!token) {
    wx.showModal({
      title: '请先获取 Token',
      content: 'IDE Console 粘贴这段拿 token:',
      confirmText: '看代码',
      success: () => wx.showModal({
        title: '复制这段到 Console',
        content: 'wx.request({url:"https://43.139.176.199/api/test/dev-reissue",method:"POST",header:{"Content-Type":"application/json"},data:{userId:2},success:(res)=>{if(res.data.code===0){wx.setStorageSync("token",res.data.data.token);wx.setStorageSync("user",{id:2,openid:res.data.data.openid,nickname:res.data.data.nickname,avatar:null});console.log("TOKEN OK")}else{console.error("FAIL",res.data)}}})',
        showCancel: false,
      }),
    });
    return;  // 不跑 submit
  }
  // ... 原有逻辑
}
```
