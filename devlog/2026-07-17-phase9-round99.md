## 关键代码

### AI 文案 in CONSTELLATIONS
```js
{
  id: 'name', label: '姓名', type: 'text',
  ai: '嗨, 先告诉我你的名字吧?'
}
```

### 弹窗 AI 气泡
```html
<view class="modal-ai-bubble" style="border-left-color: {{modalConstColor}};">
  <view class="modal-ai-avatar" style="background: {{modalConstColor}};">✨</view>
  <view class="modal-ai-content">
    <text class="modal-ai-name">简历助手</text>
    <text class="modal-ai-text">{{modalFieldAi}}</text>
  </view>
</view>
```

### AI 气泡样式 (太空对话感)
```css
.modal-ai-bubble {
  background: rgba(255,255,255,0.04);
  border-radius: 16rpx;
  border-left: 6rpx solid;  /* 用 const 颜色 */
}
.modal-ai-avatar {
  box-shadow: 0 0 12rpx currentColor;  /* 头像发光 */
}
```
