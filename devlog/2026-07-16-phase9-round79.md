## 问题

Naive SSE:
- N admin × M tabs = N*M 个 SSE 连接
- 每连接每 10s 一次 snapshot
- DB load: `(N*M) × 6 req/min = 6*N*M req/min`
- 10 admins × 5 tabs = **300 req/min** 在 dashboard endpoints

## 解法 — Coalescing

**核心**: module-level cache + 共享 in-flight promise + 单一 ticker.

```
ticker (single setInterval):
  ↓ every 10s
getSharedSnapshot():
  ├─ cache fresh (<1s)? → return cache
  ├─ in-flight promise? → join it (no duplicate fetch)
  └─ else → kick off fetchSnapshot() → cache + return
  ↓
broadcast payload to all connections in Set
```

### 量化

| 场景 | 之前 | 之后 |
|---|---|---|
| 10 admins × 5 tabs, snapshot/10s | 300 req/min | **6 req/min** (1 snapshot × 6/min) |
| 50 connections 突发连接 (initial snapshot) | 50 并发 DB query | **1 query** (all join in-flight) |
| 0 连接 idle | — | ticker 跳过 DB |

## 设计

### Module-level state
```js
const connections = new Set(); // Set<{ res, id, openid, connectedAt }>
let cachedSnapshot = null;
let cachedSnapshotAt = 0;
let snapshotPromise = null;
```

### `getSharedSnapshot()`
```js
async function getSharedSnapshot() {
  const now = Date.now();
  if (cachedSnapshot && now - cachedSnapshotAt < 1000) return cachedSnapshot; // fresh
  if (snapshotPromise) return snapshotPromise; // join in-flight
  snapshotPromise = fetchSnapshot()
    .then(s => { cachedSnapshot = s; cachedSnapshotAt = Date.now(); return s; })
    .catch(e => { /* don't poison cache */ return cachedSnapshot || { ts: Date.now(), error: e.message }; })
    .finally(() => { snapshotPromise = null; });
  return snapshotPromise;
}
```

### Ticker (single)
```js
setInterval(async () => {
  if (connections.size === 0) return; // skip DB
  const snap = await getSharedSnapshot();
  const payload = `event: dashboard-update\ndata: ${JSON.stringify(snap)}\n\n`;
  for (const conn of connections) {
    try { conn.res.write(payload); } catch (_) { /* dead */ }
  }
}, 10_000);
```

### Per-connection lifecycle
- on connect: `connections.add({res, id, openid})`
- on `req.close` 或 `req.error`: `connections.delete(conn)`
- on heartbeat tick: same broadcast loop iterates connections

### Observability
- `_stats()` exports `{connections, lastSnapshotAt, cacheAgeMs}` for `/api/health/ready` extensions (R80+)
