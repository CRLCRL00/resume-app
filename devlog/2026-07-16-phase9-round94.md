## 关键代码

### 完成度算法 (calcCompletion)

```js
// 总权重: 基本 25 + 教育 20 + 工作 25 + 期望 20 + 技能 10 = 100
基本: name(10) + gender(5) + degree(5) + phone(5) = 25
教育: school(10) + major(5) + start+end(5) = 20
工作: company(10) + title(5) + start+end(5) + desc(5) = 25
期望: city(7) + position(7) + salary_min(3) + salary_max(3) = 20
技能: skillsCount > 0 → 10
```
