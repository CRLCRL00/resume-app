## 改了什么

| 文件 | 改动 |
|---|---|
| `mini-program/pages/form/bigscreen/bigscreen.wxml` | 3 处 `wx:key` 改 plain key (不带 `{{}}`) |
| `mini-program/pages/form/bigscreen/bigscreen.js` | onSubmit 加 `wx.showLoading` + catch 加 `console.error` |

### wx:key 修复

| before | after |
|---|---|
| `wx:key="bs-{{index}}"` | `wx:key="bs{{index}}"` |
| `wx:key="c-{{item.id}}"` | `wx:key="con{{item.id}}"` |
| `wx:key="p-{{item.id}}"` | `wx:key="p{{item.id}}"` |

**Why**: 微信 wx:key 不支持 `{{}}` 表达式 — 应是 plain string.

### showLoading 修复

```js
// before (没 showLoading, catch 直接 hide)
request(...)
  .then(...)
  .catch(() => wx.hideLoading());  // hide 计数可能错位

// after
wx.showLoading({ title: '...', mask: true });
request(...)
  .then(...)
  .catch((err) => {
    wx.hideLoading();
    console.error('submit failed:', err);  // debug info
  });
```
