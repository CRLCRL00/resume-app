### 2. 技能雷达图 (R118 T2, 脉脉风格个人画像)

```js
function _calcRadarScores(form, skillsCount) {
  // 5 维 0-100 分: basic / education / work / expected / skills
  const score = (obj, fields) => {
    let filled = 0;
    for (const f of fields) if (obj[f] && String(obj[f]).trim()) filled++;
    return Math.round((filled / fields.length) * 100);
  };
  return {
    basic: score({...}, ['name', 'gender', 'degree', 'phone']),
    education: score(educations[0], ['school', 'major', 'start', 'end']),
    work: score(experiences[0], ['company', 'title', 'start', 'end', 'desc']),
    expected: score(expected, ['city', 'position', 'salary_min', 'salary_max']),
    skills: skillsCount >= 3 ? 100 : skillsCount >= 1 ? 60 : 0,
  };
}
```

### Nit #1: 测试不匹配 (R104 → R116 翻转)

**问题**: R116 翻转测试 `assert !src.includes('type="2d"')` 因为没 canvas, 但 R118 T2 加了 type=2d radar canvas, 测试 fail.

**解决**: 翻转测试改为正向断言 `assert src.includes('type="2d"')` (R118 翻转 R116: 大屏雷达图复用 type=2d canvas).
