## R91-B: deprecation fix

3 处 `wx.getSystemInfoSync` → `wx.getDeviceInfo` / `wx.getWindowInfo`:

| 文件 | before | after |
|---|---|---|
| `mini-program/app.js:58-60` | `wx.getSystemInfoSync().platform` | `wx.getDeviceInfo().platform` |
| `mini-program/utils/monitor.js:14` | `wx.getSystemInfoSync().platform` | `wx.getDeviceInfo().platform` |
| `mini-program/admin/pages/dashboard/dashboard.js:35` | `wx.getSystemInfoSync()` (window+platform) | `wx.getWindowInfo() + wx.getDeviceInfo()` |

新 API:
- `wx.getDeviceInfo()` → platform / model / system
- `wx.getWindowInfo()` → windowWidth / windowHeight / pixelRatio
- `wx.getAppBaseInfo()` → app version / SDK version
- `wx.getAppAuthorizeSetting()` → auth scope
- `wx.getSystemSetting()` → system theme / bluetooth

旧 `getSystemInfoSync` 返回混合, 但 3.16.1 起标 deprecated.
