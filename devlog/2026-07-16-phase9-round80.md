## 5 Metrics added

| `sse_cache_age_seconds` | Gauge | — | set on cache hit + after fetch |

## 设计

### Singleton pattern (跟 metrics.js 一致)
```js
const sseActiveConnections = globalThis.__sseActiveConnections
  || new client.Gauge({ name: 'sse_active_connections', help: '...' });
globalThis.__sseActiveConnections = sseActiveConnections;
```
避免 require 循环导致 double-register.

### 集成点

| 事件 | 指标 |
|---|---|
| on `connections.add(conn)` | `sse_active_connections.set(size)` + `sse_connections_total.inc()` |
| on `connections.delete(conn)` | `sse_active_connections.set(size)` |
| in `fetchSnapshot` (success) | `sse_snapshot_duration.observe(ms/1000)` |
| in `getSharedSnapshot` (cache hit) | `sse_cache_age.set(seconds_old)` |
| in `getSharedSnapshot` (after fetch) | `sse_snapshots_total.inc()` + `sse_cache_age.set(0)` |
