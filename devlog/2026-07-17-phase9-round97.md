## 设计 — AI 对话式

**核心**: 不用 form / input 卡片 / 步骤条, 用 chat-like 气泡:
- AI 提问 (左灰气泡 + ✨ avatar)
- 用户答 (右绿气泡 + 我 avatar)
- 底部固定输入栏 (大输入框 + 发送按钮)
- 浮动预览卡片 (≥1024 屏, sticky)

**5 段对话脚本** (13 节点):
```
Step 0 基本信息 (4 节点):
  1. 名字?
  2. 性别? (chips)
  3. 学历? (picker)
  4. 手机 (optional, 可跳过)

Step 1 教育 (3 节点):
  5. 学校专业?
  6. 起止时间? (dateRange)
  7. + 还有其他教育? (addMore)

Step 2 工作 (4 节点):
  8. 公司职位? (work)
  9. 时间段? (dateRange)
  10. 工作描述? (textarea)
  11. + 还有其他工作? (addMore)

Step 3 期望 (3 节点):
  12. 城市?
  13. 岗位?
  14. 薪资范围? (salaryRange)

Step 4 技能 (1 节点):
  15. 技能列表? (textarea)
```

## 关键代码

### 状态机
```js
CHAT_SCRIPT = [{step, stepName, ai, field, type, options?}]
chatStep 推进 → _pushAI (push AI bubble + 设当前 input type)
用户答 → _submitAnswer (push user bubble + apply form + advance)
```

### 气泡样式 (革命性 vs 传统)
```css
.msg-bubble.ai {
  background: #fff;
  border-bottom-left-radius: 8rpx;  /* 尖头 */
  font-size: 30rpx;
}
.msg-bubble.user {
  background: linear-gradient(135deg, #07c160, #00a854);
  color: #fff;
  border-bottom-right-radius: 8rpx;
}
.msg-bubble.current {
  animation: pulse 1.5s infinite;  /* 当前问题呼吸效果 */
}
```

### 输入栏 (底部固定, 像微信聊天)
```css
.chat-input-bar {
  position: fixed;
  bottom: 0; left: 0; right: 0;
  background: rgba(255,255,255,0.95);
  backdrop-filter: blur(20rpx);
}
```

### 多类型问题 (5 种)
- `text`: input
- `textarea`: 多行
- `chips`: 大按钮 (性别)
- `picker`: 学历选择
- `dateRange`: 起止时间
- `work`: 公司·职位 单行
- `salaryRange`: 下限-上限
- `addMore`: 教育/工作重复
