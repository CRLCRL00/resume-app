## 起点

R99 用 `bs-{{index}}` fail (微信不支持 `{{}}`). R100 改 `bs{{index}}` 仍 fail. R101 改 `*this` fail (item 是 plain object, `[object Object]` 不 unique). R102 用 item 字段.

## 微信 wx:key 实际规则 (官方)

接受:
- **`*this`** — 仅当 item 是 primitive 或 unique object (我们 item 是 plain object, 不 unique)
- **字符串字面量 (item 字段名)** — `wx:key="id"`, `wx:key="value"` → 编译器找 `item.id` / `item.value` 当 key
- 字符串/数字字面量 (写死)

不接受:
- `{{}}` 表达式

## 修

| before | after |
|---|---|
| `wx:key="*this"` (backgroundStars) | `wx:key="x"` (item.x unique) |
| `wx:key="*this"` (constellations) | `wx:key="id"` (item.id 是 basic/edu/...) |
| `wx:key="*this"` (particles) | `wx:key="id"` (item.id 是 field name) |
| `wx:key="mc-{{index}}"` (modal chips) | `wx:key="value"` (item.value 是 male/female/other) |
