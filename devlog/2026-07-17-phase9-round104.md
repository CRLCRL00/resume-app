# 开发日志 — 2026-07-17（Phase 9 Round 104 — Canvas type=2d 升级）

## 起点

R103 真机 preview 时 console:
> `[pages/form/bigscreen/bigscreen] [Component] <canvas>: canvas 2d 接口支持同层渲染且性能更佳，建议切换使用。`

R103 用了旧 `<canvas>` + `wx.createCanvasContext()` API. 微信推荐新 API:
- 旧: `<canvas canvas-id="x">` + `wx.createCanvasContext('x')` + ctx.draw()
- 新: `<canvas type="2d" id="x">` + `wx.createSelectorQuery().select('#x').node()` + canvas.getContext('2d') (Web 标准)

## 改了什么

### wxml
```html
<!-- before -->
<canvas canvas-id="starfield-lines" class="lines-canvas" .../>

<!-- after -->
<canvas type="2d" id="starfield-lines" class="lines-canvas" .../>
```

### js — _drawLines 重写
```js
// before (旧 API)
_drawLines(width, height, dpr = 2) {
  const ctx = wx.createCanvasContext('starfield-lines');
  ctx.setLineWidth(2);
  ctx.setStrokeStyle(...);
  // ... draw ...
  ctx.draw();  // commit
}

// after (新 API)
_drawLines(width, height) {
  const query = wx.createSelectorQuery();
  query.select('#starfield-lines').fields({ node: true, size: true }).exec((res) => {
    const canvas = res[0].node;
    const ctx = canvas.getContext('2d');
    ctx.lineWidth = 2;
    ctx.strokeStyle = ...;
    // ... draw, 立即 commit (no .draw())
  });
}
```

**注意**:
- 旧 API 用 `setLineWidth/setStrokeStyle/beginPath/moveTo/lineTo/stroke/draw`
- 新 API 用 `lineWidth/strokeStyle/beginPath/moveTo/lineTo/stroke` (Web 标准)
