## 7 panels

| # | 面板 | 指标 | 告警阈值 |
|---|---|---|---|
| 1 | Active SSE Connections (stat) | `sum(sse_active_connections)` | >20 yellow, >50 red |
| 2 | Connections / sec | `rate(sse_connections_total[5m])` | spike 监控 |
| 3 | Snapshot Generation / sec | `rate(sse_snapshots_total[5m])` | ≈ 0.1 (1/10s/pod) |
| 4 | Snapshot Duration p50/p95/p99 | `histogram_quantile` | p95 > 1s 慢 |
| 5 | Rejected / sec by reason | `rate(..._total) by (reason)` | 看 per-admin cap 触发 |
| 6 | Replayed Events / sec | `rate(sse_replayed_events_total[5m])` | 高 = 客户端频繁断 |
| 7 | Snapshot Cache Age (stat) | `max(sse_cache_age_seconds)` | >5s yellow, >10s red |

## Redis buffer (用 prometheus-redis-exporter)

| PromQL | 含义 |
|---|---|
| `redis_list_length{key="sse:replay:buffer"}` | 当前条数 (≤100) |
| `redis_key_ttl_seconds{key="sse:replay:buffer"}` | TTL (应 ≈ 86400) |
| `redis_value_value{key="sse:event:id"}` | event id counter |
