## 起点

R98 粒子星图骨架已搭, 但粒子静态. R103 加 2 项 polish:
- **连线** (filled 粒子之间画 SVG/Canvas 线)
- **漂浮** (粒子 CSS 动画 sin/cos 模拟)

## 改了什么

### Canvas 连线
- `bigscreen.wxml` 加 `<canvas canvas-id="starfield-lines" disable-scroll="true">` 覆盖全屏
- `bigscreen.js` 加 `_drawLines(width, height, dpr)` 方法
  - 收集所有 filled 粒子 (按星座 color)
  - 两两连线 (距中心越近越亮, alpha 1 - dist/maxDist)
  - 中心 → 每个 filled 粒子的连线 (毛笔效应)
  - 用 `setStrokeStyle` + `beginPath` 画
- `pointer-events: none` 不拦截粒子点击
- 触发点: `onLoad`, `onShow`, `_saveModal` (字段填后)

### 粒子漂浮
- `bigscreen.wxss` `.particle` 加 `@keyframes float` 4s 无限循环
- 5 个错位延迟 (`nth-child` animation-delay) 看起来独立
- `transform: translate` 不影响 click

## 关键代码

### drawLines 核心
```js
_drawLines(width, height, dpr = 2) {
  const ctx = wx.createCanvasContext('starfield-lines');
  const filledPoints = [];
  for (const c of this.data.constellations) {
    for (const p of c.particles) {
      if (this._isFieldFilled(p.id)) {
        filledPoints.push({ x: p.x * dpr, y: p.y * dpr, color: c.color, rgb: c.colorRgb });
      }
    }
  }
  // 中心 → 每个粒子 (毛笔)
  ctx.setLineWidth(2);
  for (const p of filledPoints) {
    ctx.setStrokeStyle(`rgba(${p.rgb}, 0.6)`);
    ctx.beginPath();
    ctx.moveTo(w/2, h/2);
    ctx.lineTo(p.x, p.y);
    ctx.stroke();
  }
}
```

### float animation
```css
@keyframes float {
  0%, 100% { transform: translate(0, 0); }
  25% { transform: translate(2rpx, -4rpx); }
  50% { transform: translate(0, -8rpx); }
  75% { transform: translate(-2rpx, -4rpx); }
}
.particle:nth-child(2) { animation-delay: 0.5s; }
```
