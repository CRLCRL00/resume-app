### Step 5: pm2 start (试错)
**问题 1**: `pino-pretty` MODULE_NOT_FOUND
- 原因: ecosystem.config.js dev env 走 pretty mode, `--omit=dev` 没装
- 解决: `--env production` → JSON line mode

**问题 2**: EADDRINUSE 3000
- 原因: aigc-web (pm2 id 1) 占着 3000
- .env PORT=3000 → 改 3003

**问题 3**: Redis NOAUTH
- 原因: `.env` 中 `REDIS_PASSWORD=` (空 value)
- server Redis `requirepass ResumeRedis@2026` 已设 (我测过 redis-cli -a)
- 解决: pm2 inline env `REDIS_PASSWORD='ResumeRedis@2026'`

**问题 4**: MySQL root access denied
- 原因: `.env` DB_USER=root, DB_PASSWORD=错的 (sync 过来的旧值)
- 测: `resume_app_user` 用 `27a25774642b01bcd15d0ddeaf200069` 连得上
- 解决: `DB_USER=resume_app_user` + inline DB_PASSWORD + 改 .env DB_USER
