# 服务器域名白名单配置

填在微信小程序管理后台 → 开发 → 开发设置 → 服务器域名。

## 当前（开发版 / 体验版）

```
request 合法域名:    https://43.139.176.199
uploadFile 合法域名: https://43.139.176.199
downloadFile 合法域名: https://43.139.176.199
```

## 注意事项

1. **当前 43.139.176.199 是 IP 地址 + 自签证书**
   - 微信开发版 / 体验版 / 模拟器允许 IP + 不需要校验
   - 微信开发者工具需开启「不校验合法域名」: `详情 → 本地设置 → 不校验合法域名`

2. **正式上线需**：
   - 备案过的域名（如 `crlcrl.com`）
   - CA 证书（Let's Encrypt / 阿里云 SSL）
   - 域名完成备案后 30 个自然日才能提交正式版审核

3. **业务域名**（web-view 使用）：
   - 本小程序未使用 web-view，无需配置

## 切换到生产域名时

1. 部署时将后端 `BACKEND_URL` 环境变量改为 `https://yourdomain.com`
2. 小程序前端所有 `wx.request` 调用更新 URL
3. 微信后台更新白名单
4. 上传新体验版重新审核
