## 二、备案问题回答

### ❓ "小程序要备案吗？"

**要，但只在「正式上线」前备案。**

**体验版 / 开发版 / 审核版**：不需要备案。  
**发布正式版（生产环境）**：必须 ICP 备案。

> 备案 = 「ICP 备案」= 工信部要求，由**服务器域名**触发。

---

### 📋 两种备案概念（容易混）

| 概念 | 谁负责 | 何时需要 |
|------|--------|----------|
| **ICP 备案**（域名备案） | 你（小程序主体）或个人 | 域名正式使用前 |
| **小程序备案** | 微信平台（mp.weixin.qq.com 自动引导） | 已合并到 ICP 备案 |

**2023-09 后**：小程序备案 = ICP 备案，一个入口（工信部）。

---

### 🔁 备案流程（顺序）

#### Step 1：买域名（可选，留 IP 不行）

- 推荐：`阿里云 / 腾讯云 / Cloudflare` 注册
- 后缀：`.com` / `.cn` / `.com.cn`
- 大约 ¥55-70/年

#### Step 2：域名 ICP 备案

- 入口：https://beian.miit.gov.cn/
- 主体类型：
  - **个人**：身份证 + 人脸核验 + 微信绑定手机
  - **企业**：营业执照 + 对公账户验证 + 法人身份证
- 资料：身份证 / 营业执照、负责人手机号、邮箱、应急联系方式
- 时长：**7-20 个工作日**（个人快，企业慢）
- 免费（除域名本身费用）

#### Step 3：公安联网备案（额外，强制）

- 工信部备案**通过后**，自动跳转公安备案
- 入口：https://beian.mps.gov.cn
- 时长：**3-7 个工作日**
- 免费

#### Step 4：DNS 解析 + CA 证书

- DNS 把域名解析到 `43.139.176.199`
- CA 证书（Let's Encrypt 免费 / 阿里云 SSL 付费）
- 域名解析到中国大陆 IP 还需要「公安联网备案」后才能用 443

#### Step 5：小程序后台填域名 + 上传审核

- 完成上面 3-4 步后
- 微信小程序管理后台 → 域名白名单 改成 `https://yourdomain.com`
- 上传新体验版，重新提交审核
- 审核时长：**1-7 天**

### 🎯 推进路径建议

#### 路径 A：先体验版上线（最快）

```
1. 微信管理后台填类目 + 域名白名单（IP）
2. 上传体验版 + 提交审核
3. 通过后 → 体验码分享给朋友/审核员
4. 期间可同步申请 ICP 备案
5. 备案通过 → 切换到域名 → 提交正式版审核
```

⏱️ 体验版上线：**3-5 天**（准备 + 审核）  
⏱️ 正式版上线：**+ 14-30 天**（备案）

#### 路径 B：跳过体验版，直接正式版

```
1. 买域名 + ICP 备案（14-30 天）
2. 公安备案（3-7 天）
3. 配置 DNS + SSL
4. 微信管理后台切换域名
5. 上传体验版 → 提交审核（1-2 天）
合计：约 30-50 天
```

不推荐：风险大、时间长。
