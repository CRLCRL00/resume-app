## 五、配置体验版

**左侧菜单 → 版本管理 → 开发版本**

> 你需要在本地「微信开发者工具」上传体验版：
> 1. 打开 IDE → 「智能简历助手」项目（已在 project.config.json 配置）
> 2. 右上 → 「上传」
> 3. 版本号：`1.0.0`
> 4. 项目备注：`Phase 7 体验版 - 隐私协议接入 - 服务条款接入`
> 5. 上传

上传完回到管理后台 → 「开发版本」会出现新版本，点 **「提交审核」**。

## 七、提交审核 + 跟进

1. 点击「**提交审核**」
2. 等待 1-7 个工作日（工具类通常 1-2 天）
3. 通知渠道：
   - 微信（绑定管理员微信会收到通知）
   - 管理后台「通知中心」
4. 通过后 → 选「**发布**」（体验版转正式版）
5. 拒绝 → 看拒绝原因 → 调整 → 重新提交
