## Admin 账号（后台管理）

**openid**：`已在 admins 表注册的 openid`

**查看当前 admins**：
```bash
mysql -u resume_app_user -p resume_app -e "SELECT * FROM admins"
```

**添加新 admin**（如审核员需）：
```sql
INSERT INTO admins (openid, note) VALUES ('审核员_openid', '审核专用');
```
